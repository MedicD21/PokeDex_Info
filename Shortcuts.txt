Navigation Shortcuts

ll          → List files in current directory
la          → List all files including hidden
clr         → Clear terminal
up          → Move up one folder
up2         → Move up two folders

Git Shortcuts

gstatus     → git status
gadd .      → git add .
gcommit -m "msg" → git commit -m "msg"
gpush       → git push
gpull       → git pull
gbranch     → git branch
gcheckout X → git checkout X
glg         → Pretty git log (graph view)


Git Power Tools

gfix "msg"  → Auto stage + commit + push (1 command workflow)

gundo       → Undo last commit but keep your changes
              (git reset HEAD~1)

gsync       → Pull with rebase + push

Node, Docker, Backend AUto Detect

Node Projects:
- Detects package.json
- Installs node_modules automatically if missing

Docker:
- Detects docker-compose.yml
- Marks project as docker-enabled

Python Backend:
- Detects main.py or app.py
- Marks project as backend-ready


Env Variables

- Reads .env on entering a folder
- Sets environment variables for the session

OMPOSH

- Theme: montys.omp.json
- Shows folder, git branch, machine info, runtime, icons
- Shows Python venv when activated
- Supports Nerd Font icons

Auto Dir Change Engine

1. Detect project root
2. Load .env file
3. Auto activate Python venv
4. Auto install npm packages if needed
5. Detect Docker project
6. Detect backend script
7. Print “Project detected at …”
